# Translate-VRC-MMD
Translates blendshapes to their japanese mmd versions

---
**Quick one time use: [https://pastebin.com/raw/7Pncz1xY](https://pastebin.com/raw/7Pncz1xY)**

- Select all, copy
In blender:
- Select object with shapekeys
- Open a text editor and press "+ New"
- Paste code
- Run (Play button)
